# NutriGame Food Detection

Food object detection using YOLOv8n model. Detects 80 different object classes from the COCO dataset.

## Features
- Fast inference with YOLOv8n
- Visual bounding box display
- Confidence scores
- Support for multiple food items

## Usage
1. Upload an image
2. Wait for detection results
3. View annotated image and detected objects

## Model
- **YOLOv8n**: Ultralytics nano model
- **Dataset**: COCO (80 classes)
